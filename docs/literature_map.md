# Literature Map

## Main Buckets

### Spatially Structured Uncertainty

Purpose: support the argument that uncertainty should be analyzed by spatial
location and structure, not only by global average.

### Failure Prediction and Quality Control

Purpose: position the work as reliability and deployment auditing, not only
segmentation improvement.

### Weak and Low-Cost Annotation

Purpose: connect failure regions to low-label medical image learning.

### Frozen Foundation Features

Purpose: justify optional use of DINOv2, UNI, or SAM image encoders as feature
extractors for ambiguity or confounder-region discovery.

### Boundary, Topology, and Object-Level Metrics

Purpose: justify boundary Dice, HD95, AJI, PQ, connected components, and
topology-sensitive failure descriptors.

## Rule

References should be organized around claims, not accumulated as a flat list.
